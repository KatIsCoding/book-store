## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- npm
- A web browser

### Setup

- `git clone` this repo
- `cd` into the directory

### Install

- Execute `npm install`

### Usage

- Execute `npm start`

### Run tests

- Execute `npm run test`

### Deployment

- Copy and paste the `build/` folder into your web server
